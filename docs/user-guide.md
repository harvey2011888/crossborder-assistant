## 开始使用

### 邀请Bot到服务器

1. 获取Bot邀请链接（由管理员提供）
2. 选择要添加Bot的服务器
3. 授权必要权限
4. 完成验证

### 首次使用

在Discord频道中输入：
```
/start
```

Bot将发送欢迎消息和使用指南。

### 查看帮助

```
/help
```

或查看特定命令的帮助：
```
/help search
```

## 基础命令

### /help - 帮助命令

显示所有可用命令或特定命令的详细帮助。

**用法：**
```
/help [命令名]
```

**示例：**
```
/help
/help search
/help order
```

### /start - 开始使用

显示欢迎消息和快速入门指南。

**用法：**
```
/start
```

### /settings - 用户设置

查看和修改个人设置。

**用法：**
```
/settings
/settings ai gemini    # 切换AI提供商
/settings language en  # 切换语言
```

### /ai_switch - 切换AI

切换不同的AI服务提供商。

**用法：**
```
/ai_switch [gemini|qianwen|openai]
```

**示例：**
```
/ai_switch gemini
/ai_switch qianwen
```

## 购物功能

### /search - 搜索商品

在中国电商平台搜索商品。

**用法：**
```
/search <关键词> [平台] [页码]
```

**参数：**
- `关键词`（必需）：要搜索的商品名称
- `平台`（可选）：taobao（淘宝）或 jd（京东）
- `页码`（可选）：搜索结果页码，默认1

**示例：**
```
/search iPhone 15
/search 运动鞋 taobao
/search 手机壳 jd 2
```

**搜索结果说明：**
- 显示商品标题、价格、平台、销量
- 点击标题可查看商品详情
- 使用按钮进行收藏、分享、创建订单

### /recommend - 智能推荐

根据您的偏好和历史记录推荐商品。

**用法：**
```
/recommend [类别]
```

**示例：**
```
/recommend
/recommend 电子产品
/recommend 服装
```

### /compare - 商品对比

对比多个商品的价格、评分、销量等信息。

**用法：**
```
/compare <商品链接1> <商品链接2> [商品链接3]
```

**示例：**
```
/compare https://item.taobao.com/xxx https://item.jd.com/xxx
```

### /translate - 翻译商品信息

将商品信息翻译成您选择的语言。

**用法：**
```
/translate <商品链接或描述>
```

**示例：**
```
/translate https://item.taobao.com/xxx
/translate 这款手机采用了最新的处理器
```

## 订单管理

> **注意：** 订单功能需要对接自建跨境电商平台API，当前为预留框架。

### /order create - 创建订单

从商品链接创建代购订单。

**用法：**
```
/order create <商品链接> [数量] [规格]
```

**示例：**
```
/order create https://item.taobao.com/xxx
/order create https://item.jd.com/xxx 2 黑色XL
```

**流程：**
1. 发送商品链接
2. Bot获取商品信息并计算价格
3. 确认订单详情
4. 选择收货地址
5. 生成订单并跳转支付

### /order list - 订单列表

查看您的所有订单。

**用法：**
```
/order list [状态] [页码]
```

**参数：**
- `状态`（可选）：pending（待支付）、paid（已支付）、shipped（已发货）、delivered（已送达）
- `页码`（可选）：默认1

**示例：**
```
/order list
/order list pending
/order list shipped 2
```

### /order status - 查询订单状态

查询特定订单的详细状态。

**用法：**
```
/order status <订单号>
```

**示例：**
```
/order status ORD202401010001
```

### /order cancel - 取消订单

取消未发货的订单。

**用法：**
```
/order cancel <订单号> [原因]
```

**示例：**
```
/order cancel ORD202401010001
/order cancel ORD202401010001 不想要了
```

## 物流查询

### /shipping - 运费估算

估算商品寄送到指定国家的运费。支持30+个目的地国家，自动计算多条物流线路的价格和时效。

**用法：**
```
/shipping weight:<重量(克)> destination:<目的地国家> [length:<长度(cm)>] [width:<宽度(cm)>] [height:<高度(cm)>]
```

**参数：**
- `weight`（必需）：包裹重量，单位**克**
- `destination`（必需）：目的地国家（从列表中选择）
- `length`（可选）：包裹长度（厘米），默认10
- `width`（可选）：包裹宽度（厘米），默认10
- `height`（可选）：包裹高度（厘米），默认10

**示例：**
```
/shipping weight:100 destination:美国
/shipping weight:500 destination:日本 length:15 width:10 height:8
/shipping weight:2000 destination:德国
```

**返回信息：**
- 📦 包裹信息（重量、尺寸、体积）
- 🚚 可用物流线路（最多显示5条）
  - 线路名称和类型（标准/特快）
  - 运费价格（含操作费）
  - 运输时效（天数）
  - 历史使用次数
  - 送达率百分比
  - 计费方式（实重/体积重）
  - 标签信息（可投保、免税/VAT等）
- ⚠️ 不可用线路及原因

**支持的国家：**
美国、加拿大、英国、德国、法国、澳大利亚、日本、韩国、新加坡、马来西亚、新西兰、意大利、西班牙、荷兰、比利时、奥地利、瑞士、瑞典、挪威、丹麦、芬兰、爱尔兰、葡萄牙、希腊、波兰、捷克、匈牙利、罗马尼亚、保加利亚等30+国家。

**计费说明：**
- **实重计费**：按包裹实际重量计算运费
- **体积重计费**：按长×宽×高/6000计算体积重量，取实际重量和体积重量的较大值
- 价格包含运费和操作费
- 部分线路支持购买运输保险（可投保标签）

**注意事项：**
- 以上价格为预估价格，实际价格以订单确认时为准
- 送达率表示历史包裹在预计时间内送达的百分比
- 部分线路可能有重量限制（如最低200g）

### /track - 包裹追踪

**返回信息：**

## 交互功能

### 商品卡片

搜索商品后，每个商品会显示一个卡片，包含：
- **查看详情**：点击标题跳转到商品页面
- **创建订单**：快速创建代购订单
- **收藏**：保存到收藏夹
- **分享**：分享给其他用户

### 订单操作

查看订单时，根据订单状态显示不同操作：
- **待支付**：支付、取消订单
- **已发货**：追踪物流
- **已送达**：确认收货、查看详情

### 分页浏览

当结果较多时，使用分页按钮：
- **首页**：跳转到第一页
- **上一页**：返回上一页
- **页码指示器**：显示当前页/总页数
- **下一页**：进入下一页
- **末页**：跳转到最后一页

## 常见问题

### Q: Bot没有响应我的命令？

A: 请检查：
1. Bot是否在线（查看成员列表）
2. 命令前缀是否正确（默认使用 `/`）
3. 是否有权限使用该命令
4. 尝试 @Bot 直接对话

### Q: 搜索不到商品？

A: 可能原因：
1. 关键词过于具体，尝试简化
2. 平台API限制或故障
3. 商品已下架

建议：尝试不同的关键词或在不同平台搜索。

### Q: 如何切换AI提供商？

A: 使用以下命令：
```
/ai_switch gemini     # Google Gemini
/ai_switch qianwen    # 阿里千问
/ai_switch openai     # OpenAI GPT-4
```

### Q: 订单创建失败？

A: 订单功能需要对接自建平台API，当前为预留框架。请联系管理员了解具体接入时间。

### Q: 物流查询不到信息？

A: 可能原因：
1. 运单号输入错误
2. 物流公司选择错误
3. 包裹刚发货，信息未同步
4. 包裹追踪功能待平台API接入（运费测算功能已可用）

### Q: 运费测算显示"服务不可用"？

A: 请检查：
1. Bot管理员是否已配置运费API
2. 联系管理员确认 `SHIPPING_API_URL` 环境变量已设置

### Q: 为什么有些线路显示"不可用"？

A: 线路不可用可能原因：
1. 包裹重量不符合线路要求（如低于最低重量限制）
2. 包裹尺寸超出线路限制
3. 该线路暂停服务

### Q: 实重计费和体积重计费有什么区别？

A: 
- **实重计费**：按包裹实际称重重量计算运费
- **体积重计费**：按长×宽×高/6000计算体积重量，与实际重量比较取较大值计费
- 体积大但重量轻的包裹（如羽绒服、枕头）可能按体积重计费

## 提示与技巧

1. **使用自然语言**：Bot支持自然语言对话，直接描述您的需求
2. **收藏商品**：看到喜欢的商品先收藏，方便后续比较
3. **对比功能**：使用 `/compare` 对比多个商品，做出最佳选择
4. **关注时效**：不同物流方式价格和时效差异大，根据需求选择
5. **保存订单号**：订单号是重要的查询凭证，请妥善保存
